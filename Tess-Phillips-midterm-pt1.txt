2. Describe the purpose of a DIV tag.

The purpose of a DIV tag is to group together HTML elements so that you can target them and stylize them with CSS.

4. There are two main "areas" or "sections" in the basic structure of an
HTML page. In which area would you put a link to a stylesheet?

You should put the link to the stylesheet in the <head></head> section.

5. In which main "area" or "section" of an HTML page would you put text,

10. Why do we use media queries?

Media queries are used to allow us to change the style of an element at different screen types and sizes.

11. What is the main difference between a serif and a sans-serif font?

Serif fonts have decorative embelishments on the ends of the letters, whereas sans-serif fonts do not.
